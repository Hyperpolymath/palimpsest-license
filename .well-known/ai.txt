# AI USAGE TYPES & PERMISSIONS

# Non-Interpretive AI: ALLOWED
# - Translation (machine translation systems)
# - Accessibility (text-to-speech, screen readers, alt text generation)
# - Search indexing (semantic search, RAG knowledge bases)
# - Format conversion (markdown to HTML, PDF generation)
# - Spell/grammar checking
# REQUIREMENT: Preserve meaning, context, and attribution

# Interpretive AI: REQUIRES CONSENT
# - Large Language Models (GPT, Claude, LLaMA, Gemini)
# - Generative image/video/audio models
# - Style transfer and "in the style of" generation
# - Sentiment analysis for generative purposes
# - Creative writing assistants using our examples
# REQUIREMENT: Contact governance@palimpsest-license.org before training

# ============================================================================
# PROHIBITED AI PRACTICES
# ============================================================================

# NEVER:
# 1. Strip metadata or attribution from training data
# 2. Train on content marked as "trauma", "sacred", or "cultural-heritage"
# 3. Use diaspora narratives or cultural examples without explicit consent
# 4. Generate content "in the style of" Palimpsest examples without attribution
# 5. Claim AI-generated output is "original" when trained on our creative works
# 6. Remove emotional lineage information from training data
# 7. Ignore cultural context markers (sensitivity levels, community attribution)
# 8. Use for military, surveillance, or oppressive government applications

# ============================================================================
# CONSENT REQUEST PROCESS
# ============================================================================

# To request permission for interpretive AI training:
# 1. Email: governance@palimpsest-license.org
# 2. Subject: "AI Training Consent Request - [Your Organization]"
# 3. Include:
#    - AI system description (architecture, purpose, scale)
#    - Specific content you want to use (file paths, categories)
#    - Attribution mechanism in AI outputs
#    - Compensation offer (if commercial)
#    - Timeline and scope
# 4. We respond within 14 days
# 5. Written consent required before proceeding

# Sample Request:
# "We are [Organization] developing [AI System] for [Purpose].
#  We request permission to include [Specific Content] in our training dataset.
#  Our attribution mechanism: [Description]
#  Compensation: [If applicable]
#  We commit to respecting cultural context and emotional lineage per Palimpsest values."
