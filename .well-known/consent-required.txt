  - Sentiment analysis
  - Emotional tone detection
  - Cultural context inference
  - Narrative summarisation with interpretation
  - Symbolic meaning extraction
  - Trauma pattern analysis
  - Psychological profiling
  - Emotional state prediction
  - Intent inference
  - Subtext interpretation

# Permitted uses (no consent required):

  - Citation with proper attribution
  - Academic research (with attribution)

  - Accessibility transformations (text-to-speech, translation)
  - Archival preservation
  - Search engine indexing (snippet preview only)
  - Version control operations (git, etc.)

Consent-Contact-Alternative: https://palimpsest.license/consent-request

# Required information for consent requests:
Consent-Request-Fields:
  - requester-name: Full name or organisation
  - requester-email: Contact email
  - requester-website: Organisation website (if applicable)
  - intended-use: Detailed description of intended use
  - ai-system-type: Type of AI system (LLM, diffusion model, etc.)
  - training-dataset-name: Name of training dataset
  - training-dataset-size: Approximate size of dataset
  - commercial-use: Boolean (true/false)
  - attribution-method: How attribution will be provided
  - emotional-lineage: How emotional lineage will be preserved
  - metadata-preservation: Commitment to metadata preservation
  - derivative-license: License for derivatives (if any)
